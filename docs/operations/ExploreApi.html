This is the API

exploreCategoryCategoryNamePostsGet

Retrieve a list of posts for a given category, such as sports, news, or entertainment.

URL

https://api.xclone.com/v1/explore/category/{categoryName}/posts

HTTP Method

GET

Response Type

PaginatedPostsResponse

Parameters

  • Path: categoryName String

    The name of the category to fetch posts for.

  • Query: limit Integer

    The maximum number of posts to return.

  • Query: page Integer

    The page number for pagination, starting from 1.
