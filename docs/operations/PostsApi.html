This is the API

postsGet

Retrieve a list of posts, with optional filtering by user ID, hashtag, and pagination.

URL

https://api.xclone.com/v1/posts

HTTP Method

GET

Response Type

PaginatedPostsResponse

Parameters

  • Query: userId Long

    Filter posts by the ID of the user who created them.

  • Query: hashtag String

    Filter posts by a specific hashtag (e.g., swagger).

  • Query: limit Integer

    The maximum number of posts to return.

  • Query: page Integer

    The page number for pagination, starting from 1.

URL

https://api.xclone.com/v1/posts

POST

Response Type

inline_response_200_18

Parameters

  • Form: content String

  • Form: type String

  • Form: parentId Long

  • Form: visibility String

postsProfileUserIdGet

Retrieve a list of posts created by a specific user, excluding replies or reposts by default.

URL

https://api.xclone.com/v1/posts/profile/{userId}

HTTP Method

GET

Response Type

PaginatedPostsResponse

Parameters

  • Path: userId Long

    The ID of the user whose profile posts to retrieve.

  • Query: include_replies Boolean

    If true, include replies made by the user in the results.

  • Query: limit Integer

    The maximum number of posts to return.

  • Query: page Integer

    The page number for pagination, starting from 1.
