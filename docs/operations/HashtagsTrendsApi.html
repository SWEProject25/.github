hashtagsHashtagPostsGet

Retrieves posts associated with a specific hashtag.

URL

https://api.xclone.com/v1/hashtags/{hashtag}/posts

HTTP Method

GET

Response Type

PaginatedPostsResponse

Parameters

  • Path: hashtag String

    The hashtag (trend name) to retrieve posts for (e.g.,

  • Query: limit Integer

    The maximum number of posts to return.

  • Query: page Integer

    The page number for pagination, starting from 1.

hashtagsSearchGet

Search for trending or common hashtags that match a query string.

URL

https://api.xclone.com/v1/hashtags/search

HTTP Method

GET

Response Type

inline_response_200_20

Parameters

  • Query: q String

    The search string to match against hashtags.

  • Query: limit Integer

    The maximum number of hashtags to return.

GET

Response Type

inline_response_200_19
