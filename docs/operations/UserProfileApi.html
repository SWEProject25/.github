usersSearchGet

Searches for users by their username or display name with a partial match.

URL

https://api.xclone.com/v1/users/search

HTTP Method

GET

Response Type

inline_response_200_3

Parameters

  • Query: q String

    The search query string.

  • Query: page Integer

    The page number for pagination, starting from 1.

  • Query: limit Integer

    The number of results to return per page.
