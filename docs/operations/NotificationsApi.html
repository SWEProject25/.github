This is the API

usersUserIdNotificationsGet

Retrieves a list of notifications for the authenticated user.

URL

https://api.xclone.com/v1/users/{userId}/notifications

HTTP Method

GET

Response Type

inline_response_200_7

Parameters

  • Path: userId Integer

    The ID of the user requesting notifications.

  • Query: limit Integer

    The number of notifications to return.
