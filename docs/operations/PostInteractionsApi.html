This is the API

postsPostIdLikesGet

Retrieves a list of user IDs who have liked the specified post.

URL

https://api.xclone.com/v1/posts/{postId}/likes

HTTP Method

GET

Response Type

inline_response_200_21

Parameters

  • Path: postId Long

    The ID of the post to get likers for.

  • Query: limit Integer

    The maximum number of likers to return.

  • Query: page Integer

    The page number for pagination, starting from 1.

postsPostIdLikesPost

Toggles a like for the specified post by the authenticated user. If liked, it unlikes; if unliked, it likes.

URL

https://api.xclone.com/v1/posts/{postId}/likes

HTTP Method

POST

Response Type

inline_response_200_22

Parameters

  • Path: postId Long

    The unique identifier of the post to like/unlike.

postsPostIdRepostsGet

Retrieves a list of user IDs who have reposted the specified post.

https://api.xclone.com/v1/posts/{postId}/reposts

HTTP Method

GET

Response Type

inline_response_200_23

Parameters

  • Path: postId Long

    The ID of the post to get reposters for.

  • Query: limit Integer

    The maximum number of reposters to return.

  • Query: page Integer

    The page number for pagination, starting from 1.

postsPostIdRepostsPost

Toggles a repost entry for the specified post by the authenticated user. If already reposted, it removes the repost; otherwise, it creates one.

URL

https://api.xclone.com/v1/posts/{postId}/reposts

HTTP Method

POST

Response Type

inline_response_200_24

Parameters

  • Path: postId Long

    The ID of the post to repost/un-repost.
