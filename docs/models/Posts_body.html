Posts_body

  • userId : Long
    The ID of the user creating the post.
  • content : String
    The main text content of the post.
  • type : String
    The type of post (e.g., original post, reply, quote).
    Enum:
    POST
    REPLY
    QUOTE
  • parentId : Long
    The ID of the parent post if this is a reply or quote. Required if type is REPLY or QUOTE.
  • visibility : String
    Who can see the post.
    Enum:
    EVERY_ONE
    FOLLOWERS
    MENTIONED
  • mediaFiles : List
    One or more media files (images, videos) to attach to the post.
